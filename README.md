## Run Locally

Clone the project

```bash
  git clone https://github.com/mak-1997/fragile-hot-3247.git
```

Go to the project directory

```bash
  cd fragile-hot-3247/mytheresa
```

Install dependencies for frontend as well as backend

```bash
  npm install
```


Start the localhost server

```bash
  npm start
```
